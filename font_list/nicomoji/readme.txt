■フォントの概要

「ニコニコ」と「NiCONiCO」の文字をデザインをベースに、他の文字をデザインしてフォント化しました。
※このフォントはニコニコ動画公式とは無関係です

フォント名：「ニコモジ」
収録文字：カタカナ・アルファベット大文字・アルファベット小文字・数字
※ひらがな・漢字・記号等は含まれていません

フォント形式：TrueTypeフォント（プロポーショナルフォント）
動作環境：Windows 95以降、Mac OS X 10.3以降
